## Requirements
* Node.js

## Structures
```
❯ tree -d -L 2 -I node_modules
.
├── src
└── web
    ├── public
    └── src
```

* src : Server Root
* web : Web Root
  * Web 쪽을 개발하시려면 IDE 에서 web directory 를 여세요
